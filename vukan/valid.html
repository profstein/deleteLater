More Information: about this page

Learning to validate a page is an important skill. While learning all of the HTML rules takes time the process is relatively simple.

  1. Validate the page (Use a tool like the Validity extension above).
  2. Fix the first error that appears.
  3. Validate the page again/
  4. Repeat until there are no errors.

Learning the semantics takes more time. When a page is semantic it means the correct elements are around the content based on the meaning of the content. For example, paragraphs go in p elements, lists in ul or ol or dl elements, tables in the table element etc.

The basics you can learn quickly, the sectioning elements above and some of the less-used elements take time to learn.

For a video tutorial on semantic sectioning elements view this Lynda.com video

In order to get the semantics right you must first read the content and understand its meaning.
